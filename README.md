# Linear Regression with Machine Learning

An introductory machine learning project demonstrating how linear regression learns relationships between numerical features and produces continuous predictions.

## Workflow

- Load and inspect the dataset
- Prepare input and target values
- Split data for training and evaluation
- Fit a linear regression model
- Generate and inspect predictions

## Technology

Python · pandas · NumPy · scikit-learn · Jupyter Notebook

## Usage

Open the notebook in Jupyter and run the cells in sequence to follow the complete regression workflow.
